# 🤖 Agentic AI Starter Kit

Enterprise-style starter project demonstrating how modern AI agents can move beyond simple chatbot interactions into structured reasoning, workflow planning, and intelligent enterprise decision support.

---

# 🚀 Project Overview

This project showcases a foundational **Enterprise Agentic AI workflow** capable of:

* Understanding business problems
* Identifying automation opportunities
* Recommending tools and workflows
* Generating execution plans
* Evaluating operational risks
* Estimating confidence before execution

The application demonstrates how Prompt Engineering, LLM reasoning, structured outputs, and workflow intelligence can work together inside a practical AI system.

# ▶️ Setup Instructions

## 1. Clone Repository

```bash
git clone https://github.com/lavanya1402/agentic-ai-starter-kit.git
cd agentic-ai-starter-kit
```

---

## 2. Create Virtual Environment

```bash
py -3.10 -m venv venv
```

Activate:

```bash
venv\\Scripts\\activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Configure Environment Variables

Create `.env`

```env
OPENAI_API_KEY=your_api_key_here
OPENAI_MODEL=gpt-4o-mini
```

---

## 5. Run Application

```bash
streamlit run app.py
```
